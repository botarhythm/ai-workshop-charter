# ai-workshop-charter

AIを使った開発を「日雇いの天才を毎回呼ぶ」働き方から、**賢さ・段取り・品質・お金の規律を備えた"まわり続ける工房"**に変えるための運用ルールの解説サイトです。

土台は [akiratsukakoshi/fable_orchestra](https://github.com/akiratsukakoshi/fable_orchestra)。その上に私たち独自の拡張（F1/F2フェーズ制・P0直営モード・承認ゲート強化）を重ねています。

## ページ

- **[運用憲章（index.html）](https://botarhythm.github.io/ai-workshop-charter/)** — 非エンジニア向けの図解。何が変わったかを1枚で。
- **[技術解説（technical.html）](https://botarhythm.github.io/ai-workshop-charter/technical.html)** — アーキテクチャ・パターン判定・モデル役割分担・他Skills連携・導入手順。

## これは何でない

このリポジトリは**方法論の解説のみ**を含みます。個別プロジェクトの業務データ・機密情報は一切含みません。
